# Приложение "AlgEdu"

## Технологический стек:
- C++: стандарт ISO C++ 14
- Imgui: v1.91.9

## Инструкция по настройке проекта:
1. Склонировать проект.
2. Открыть папку проекта и перейти в папку с файлами данного приложения:
   ```bash
   cd "application"
   ```
3. Открыть файл application.sln в Visual Studio 2022
