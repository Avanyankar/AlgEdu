AlgEdu Documentation

Complete guide to AlgEdu commands and functionality

1. Solve Linear Equation

Solves a linear equation of the form ax + b = 0.

Example Code:

solve_linear(2, -4)  # 2x - 4 = 0

Output:

Solution: x = 2.0

2. Quadratic Equation Solver

Finds roots of quadratic equation ax² + bx + c = 0.

Example Code:

solve_quadratic(1, -5, 6)  # x² -5x + 6 = 0

Output:

Roots: x₁ = 3.0, x₂ = 2.0

3. Matrix Multiplication

Multiplies two matrices and returns the result.

Example Code:

A = [[1, 2], [3, 4]]
B = [[5, 6], [7, 8]]
matrix_multiply(A, B)

Output:

Result: [[19, 22], [43, 50]]

4. Derivative Calculator

Calculates the derivative of a given function.

Example Code:

f = "x^3 + 2x^2 - 5x + 1"
derivative(f)

Output:

Derivative: 3x^2 + 4x - 5

5. System of Equations Solver

Solves a system of linear equations.

Example Code:

# 2x + y = 5
# x - y = 1
equations = [
    [2, 1, 5],
    [1, -1, 1]
]
solve_system(equations)

Output:

Solution: x = 2.0, y = 1.0
